# 30 Days of Python — My Solutions

My solutions to the exercises from
[**30-Days-Of-Python**](https://github.com/asabeneh/30-Days-Of-Python) by
[Asabeneh Yetayeh](https://github.com/asabeneh).

I'm a working developer learning Python to sharpen my skills. This repo holds my
own solutions — one folder per day. The exercises and curriculum belong to the
original author; all credit goes to him.

## Structure

```
day-NN-topic/
└── solution.py   # my solution
└── notes.md      # optional — only when something is worth writing down
```
